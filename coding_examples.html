        flex-grow: 3;
        @media (min-width: $break-l) {

        }
    }
        &:last-child::after{
            content: "";
            position: relative;
            display: none;
        }
    }
}

What is this?

For part of the task I had to make a dropdown list that could only appear at certain screensizes and would only reveal itself when hovered over by the mouse. The code above is written in CSS and shows how the dropdown list and it's list items should behave.

The dropdown contained 6 seperate list's with 5 to 6 items in each list so instead of targetting the items individually, I made my selector's target their positions conceptually. The First/ Last & nth child selectors will search for the items based on their postion relative to their parent container.

Using this technique came in very handy because if I ever needed to repeat the process all I would have to do was give another list the dropdown class.
